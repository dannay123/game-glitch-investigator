## 4. What did you learn about Streamlit and state?

- How would you explain Streamlit "reruns" and session state to a friend who has never used Streamlit?
  + Rerun is whole script run top-to-bottom. Everytime a rerun happens, everything is wiped clean, starting from 0 everytime. No memory is retained. Session state is the dict that persist across reruns. It's like a data box that is still there after countless reruns.
